This is a simple example to retrieve Orbbec Astra data with OpenNI2 

#Installation

Go to the OpenNI-Linux-x64-2.3 folder, run the install bash file

```bash
cd OpenNI-Linux-x64-2.3
chmod +x install.sh
sudo ./install.sh
```

Then 

```bash
cd ..
mkdir build
cd build
cmake ..
make
```

#To run 
go to the /bin folder
run ./viewer
